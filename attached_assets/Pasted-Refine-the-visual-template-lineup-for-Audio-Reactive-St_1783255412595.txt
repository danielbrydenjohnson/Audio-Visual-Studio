NEW TEMPLATE 1: FIBONACCI SPIRAL

Create a template based on Fibonacci spiral geometry.

Visual idea:

A deep 3D spiral made from geometric elements arranged using Fibonacci growth and golden angle spacing.

This should not be a flat 2D spiral stuck to the screen.

It should have depth, perspective, and slow organic motion.

Use a mixture of:

- Points
- Short line segments
- Small wireframe markers
- Small geometric nodes if efficient

Choose the most performant implementation that fits the existing architecture.

Requirements:

- Use the golden angle, approximately 137.5 degrees, to place elements.
- Arrange elements along one or more Fibonacci-style spiral arms.
- Give the spiral genuine z-depth.
- Give every element its own phase, speed, size, brightness, Low affinity, Mid affinity, High affinity, and random seed.
- The spiral should look intentional and recognisable when audio is paused.
- It should work well with Kaleidoscope.

Low reaction:

- Low-sensitive elements pulse outward individually along their local spiral direction.
- Some elements briefly increase in size.
- Avoid expanding the whole spiral as one rigid object.

Mid reaction:

- Mid-sensitive elements create local twisting, phase drift, and wave motion along parts of the spiral.
- Different sections should bend differently.
- Do not rotate the whole spiral as a parent group.

High reaction:

- High-sensitive elements sparkle independently.
- Brightness peaks should happen at selected nodes and line tips.
- Avoid full-canvas flashing.

Shared controls:

- Visual Density controls number of spiral elements.
- Motion Speed controls resting drift and phase motion.
- Element Size controls node or line scale.
- Depth controls z-distribution and spatial thickness.
- Glow controls luminous intensity.
- Palette controls colours.

NEW TEMPLATE 2: SACRED GEOMETRY BLOOM

Create a radial geometric template inspired by sacred geometry, flower-of-life style layouts, interlocking circles, polygons, and layered wireframe rings.

Avoid using copyrighted or overly specific iconography. Build it from general geometric constructions.

Visual idea:

Layered radial geometry made from circles, polygons, rings, and intersecting wireframe structures. It should feel ceremonial, symmetrical, and precise.

Requirements:

- Use line and wireframe geometry rather than simple point sprites.
- Build several concentric layers.
- Include repeating circular or polygonal motifs.
- Add subtle depth by placing layers at slightly different z positions.
- Each layer or element should have independent phase, speed, brightness, Low affinity, Mid affinity, High affinity, and random seed.
- The template should look strong when paused.
- It should become especially interesting under Kaleidoscope.

Low reaction:

- Low-sensitive rings or motifs pulse locally.
- Some circular forms expand or contract slightly.
- Avoid scaling all rings together.

Mid reaction:

- Mid-sensitive layers rotate or twist independently at different rates.
- Add local angular offsets and geometric breathing.
- Do not rotate the full template root as one object.

High reaction:

- Intersections, vertices, and selected edges brighten independently.
- Add short glints and flickers to changing subsets.
- Avoid full-scene strobing.

Shared controls:

- Visual Density controls number of rings, motifs, or line segments.
- Motion Speed controls layer drift and rotational phase.
- Element Size controls motif scale and line structure.
- Depth controls separation between layers.
- Glow controls line brightness.
- Palette controls colours.

NEW TEMPLATE 3: LISSAJOUS LATTICE

Create a template based on Lissajous curves and harmonic line motion.

Visual idea:

A 3D lattice of animated harmonic curves, like an oscilloscope pattern expanded into space.

This should feel mathematical, musical, and elegant.

Requirements:

- Use line geometry, not simple particles.
- Build multiple curve strands or line paths.
- Each strand should have its own frequency ratio, phase, amplitude, z offset, Low affinity, Mid affinity, High affinity, brightness, and random seed.
- The curves should move slowly even without audio.
- It should look like a designed harmonic structure when audio is paused.
- It should remain readable with Kaleidoscope enabled.

Low reaction:

- Low-sensitive strands increase amplitude or thickness impression through brightness and spacing.
- Some strands pulse outward from their own curve path.
- Avoid scaling the entire lattice.

Mid reaction:

- Mid-sensitive strands shift phase and frequency relationships.
- Add flowing curve deformation and local turbulence.
- Do not rotate the whole lattice as one group.

High reaction:

- Selected curve segments brighten or sparkle.
- Add short shimmering highlights along line paths.
- Avoid flashing all strands at once.

Shared controls:

- Visual Density controls number of curves and segments.
- Motion Speed controls harmonic animation speed.
- Element Size controls line intensity or perceived thickness where possible.
- Depth controls z spread and curve separation.
- Glow controls luminous intensity.
- Palette controls colours.
